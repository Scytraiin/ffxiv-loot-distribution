# Loot History v0.0.1-alpha

## Summary
Initial public alpha release of Loot History.

## Highlights
- Introduced the first working loot history window for tracked loot messages.
- Added searchable local history for captured loot lines.
- Added basic persistence so history can survive restarts.
- Added the first custom-repo packaging and release flow for Dalamud distribution.

## Notes
- This release focused on getting the core tracking pipeline working end to end.
- The UI and data model were still intentionally minimal at this stage.
