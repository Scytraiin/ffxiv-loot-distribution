# Loot History v0.0.2-alpha

## Summary
First major feature pass for enriched loot history.

## Highlights
- Added zone tracking so each loot entry stores where the loot happened.
- Added recipient extraction so the table can show who received the loot.
- Added recipient confidence handling for self, party/alliance, and text-only matches.
- Added debug tooling with a dedicated debug log window and session trace events.
- Reworked user-facing copy to be clearer and less prototype-oriented.
- Added plugin branding metadata and icon support.

## Notes
- This release turned the plugin from a raw line tracker into a more readable loot-history tool.
- Debug mode remained optional and developer-focused.
