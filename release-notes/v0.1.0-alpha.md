# Loot History v0.1.0-alpha

## Summary
Stability and polish update for the enriched loot tracker.

## Highlights
- Fixed loot parsing issues that could truncate captured loot lines.
- Improved raw-line handling so captured rows keep the full source text more reliably.
- Simplified internal capture and persistence behavior while preserving user-facing behavior.
- Reduced noisy internal logging and kept raw details inside debug-oriented surfaces.
- Stopped opening the debug log window automatically when debug tools are enabled.

## Notes
- This release focused on reliability and cleanup rather than major new feature scope.
