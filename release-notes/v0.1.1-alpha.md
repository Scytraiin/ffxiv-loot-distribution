# Loot History v0.1.1-alpha

## Summary
Added loot-roll awareness and improved group visibility in the main table.

## Highlights
- Added tracking for loot roll lines such as Need, Greed, and Pass.
- Correlated captured roll sessions to the final obtained loot item when possible.
- Added a `Rolls` column to the main history view.
- Added the `Group` column to the standard table with `Self`, `Party/Alliance`, and `Other`.
- Preserved normal loot rows even when no roll session was found, showing `No rolls`.
- Extended debug tracing with roll parsing and roll-correlation events.

## Notes
- Roll correlation is session-based and best-effort by item, zone, and timing window.
