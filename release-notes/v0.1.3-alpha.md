# Loot History v0.1.3-alpha

## Summary
Introduced item classification and a richer item details view.

## Highlights
- Added an `Item Details` tab alongside the main loot history view.
- Added item category resolution using `FilterGroup` with equipment refinement through `EquipSlotCategory`.
- Added persisted item classification metadata to loot records.
- Added full filter-group mapping support for known item group values.
- Made `Raw Line` a standard visible column in normal mode instead of a debug-only fallback.
- Improved the table so the same loot history can be viewed from both user-facing and metadata-focused perspectives.

## Notes
- Classification is based on local item-sheet resolution and remains best-effort for unresolved items.
