# Loot History v0.2.0-beta

## Summary
Clean-room feature expansion inspired by other loot-tracking plugins, without copying code or assets.

## Highlights
- Added item icons to the main history and item details views.
- Added rarity-aware item text styling.
- Added item hover tooltips with zone, category, equip slot, rolls, timestamp, and raw line context.
- Added item blacklist support for resolved items by item ID.
- Added richer main-window filters and a lightweight `Overview` tab.
- Added copy-to-clipboard actions for loot rows.
- Added explicit GPL-3.0 licensing and feature documentation.

## Notes
- This release focused on surfacing more of the captured item data while keeping the plugin centered on loot history.
