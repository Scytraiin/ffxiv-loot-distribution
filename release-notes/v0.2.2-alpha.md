# Loot History v0.2.2-alpha

## Summary
Cross-world recipient fix and main-window sophistication pass.

## Highlights
- Fixed recipient verification for cross-world party and alliance members.
- Added structured recipient resolution using player payloads, log entities, and party world data.
- Added world-aware recipient display in the form `First Last (World)` for cross-world players.
- Added stored loot-type bucketing with the new filters:
  - `All loot`
  - `Dungeon loot`
  - `Raid loot`
- Polished the main window with stronger filter grouping, group-aware coloring, and overview cards.
- Updated clipboard copy behavior to use the display name shown in the table.

## Notes
- Loot type classification is based on duty metadata captured at event time, so old rows keep their classification after zoning away.
