# Loot History v1.0.0-alpha

## Summary
Major UI/browser milestone release with better browsing, stronger documentation, and release-ready polish.

## Highlights
- Reworked the main window into a richer loot browser with:
  - quick filter chips
  - grouping by zone, item, or recipient
  - sort modes for time, quantity, item, and recipient
  - expandable history rows for detailed inspection
- Added favorites/pins for resolved items and a dedicated `Favorites` quick filter.
- Kept compact mode for quick monitoring while making the full window more intentional and easier to scan.
- Improved the visual direction of the main UI with a darker FFXIV-inspired palette and clearer section hierarchy.
- Added architecture documentation and flowcharts in `Architecture.md`.
- Updated the feature and README documentation so the shipped docs reflect the current browser-style UI and plugin behavior.

## Notes
- This release marks the first `1.0.0-alpha` milestone, but it is still an alpha build.
- Core loot capture remains local and read-only; the main changes in this release are browsing, usability, and documentation quality.
