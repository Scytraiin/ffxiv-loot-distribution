# Loot History v1.0.1-alpha

## Summary
Hotfix release for the new browser-style UI after the `v1.0.0-alpha` milestone.

## Highlights
- Replaced the new UI row/context identifier hashing with a safer runtime-compatible helper.
- Added a top-level UI exception guard so a window rendering failure closes the plugin windows instead of taking Dalamud down.
- Kept the `v1.0.0-alpha` browser, favorites, grouping, and compact/full-mode behavior intact.

## Notes
- This release is intended as a stability hotfix for the latest alpha UI changes.
- No loot-tracking format or data-model behavior changed in this release.
