# Loot History v1.0.2-alpha

## Summary
Follow-up crash hotfix for the new browser-style main window after the `v1.0.1-alpha` stability release.

## Highlights
- Removed the direct `ImGui.SetWindowSize(...)` calls from the main window layout flow.
- Switched compact/full-mode sizing back to the safer window-property path through the window lifecycle instead of issuing live size calls during draw.
- Kept the browser UI, grouping, favorites, overview, and compact-mode behavior introduced in `v1.0.0-alpha`.

## Notes
- This release is intended to fix the native `cimgui` crash path seen in the troubleshooting pack.
- No loot-tracking, parsing, or data-model behavior changed in this release.
