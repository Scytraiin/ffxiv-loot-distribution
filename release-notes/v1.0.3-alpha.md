# Loot History v1.0.3-alpha

## Summary
Visual refinement release for the main Loot History window, with a more FFXIV-like palette, clearer item emphasis, and improved browsing readability.

## Highlights
- Updated the main window to use a more cohesive gold-accented theme across tabs, quick filters, headers, and badges.
- Refined item rarity colors to better match familiar FFXIV-style loot expectations for common, uncommon, rare, and epic-style items.
- Reworked recipient highlighting so `Self` uses a softer gold and `Party/Alliance` uses a softer cyan instead of the previous more saturated tones.
- Added a subtle accent strip to each history row so high-value or important entries stand out faster while browsing.
- Improved quantity badges so single items stay neutral, medium stacks feel more prominent, and very large stacks are easier to spot.

## Notes
- This is primarily a presentation and usability release; loot capture and persistence behavior are unchanged.
- Compact mode, grouping, favorites, and overview behavior remain intact.
- As an alpha release, in-game feedback on readability and visual balance is still especially useful.
