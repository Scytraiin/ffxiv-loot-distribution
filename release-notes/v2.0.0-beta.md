# Loot History v2.0.0-beta

## Summary
Dalamud API 15 compatibility release built against the API 15 SDK and the local Dalamud 15.0.0.2 runtime/dev bundle, carrying forward the current structured loot and roll tracking behavior for the new game/Dalamud baseline.

## Highlights
- Updated the plugin project to `Dalamud.NET.Sdk/15.0.0`, the published SDK package for API 15.
- Updated custom repository metadata to Dalamud API level 15.
- Bumped the plugin package and assembly version to `2.0.0-beta` / `2.0.0.0`.
- Kept the existing structured chat/log capture pipeline for loot and roll messages after checking the API 15 chat/log service surface.
- Refreshed release targeting so GitHub download links point at `v2.0.0-beta`.

## Notes
- Local API 15 inspection found the plugin's critical Dalamud service entry points still available: `IChatGui.ChatMessage`, `IChatGui.LogMessage`, `ILogMessage`, `PlayerPayload`, and `ItemPayload`.
- The release build was validated against the local `15.0.0.2/dev` Dalamud bundle.
- This is a beta compatibility release for Dalamud 15. In-game validation is still recommended because loot and roll payload shapes can vary by content and localization.
